# Інструменти для роботи з датасетом (Dataset Tools)

Ця тека містить допоміжні Python-скрипти, розроблені спеціально для автоматизації процесу збору та попередньої обробки даних (Data Preprocessing) для навчання згорткової нейронної мережі.

## Вміст:

1. **`dataset_maker.py`** — утиліта для створення власної вибірки ключових точок обличчя.
2. **`merge.py`** — скрипт для безпечного злиття та фільтрації датасетів.
3. **`extract_keypoints.py`** — автоматизований конвеєр для генерації датасету емоцій на основі ключових точок.
4. **`haarcascade_frontalface_default.xml`** — каскад Хаара для автоматичної детекції облич.

---

## 1. Створення власної вибірки (`dataset_maker.py`)

Скрипт дозволяє швидко розмітити власні фотографії та згенерувати CSV-файл, повністю сумісний з форматом датасету Kaggle Facial Keypoints Detection.

### Як використовувати:
1. Запустіть скрипт вперше. Він автоматично створить теку `my_faces`.
2. Покладіть ваші фотографії (анфас) у теку `my_faces`. Переконайтеся, що назви файлів не містять кирилиці.
3. Запустіть скрипт знову. Програма автоматично знайде обличчя на фото, виріже його та відкриє вікно для розмітки.
4. Клікайте мишею по 15 ключових точках (слідкуйте за підказками у лівому верхньому куті вікна).
5. Після 15 кліків натисніть **Пробіл** (Space), щоб зберегти дані та перейти до наступного фото. 
6. Якщо потрібно перервати роботу, натисніть **ESC**. Всі розмічені до цього моменту дані будуть збережені у файлі `my_custom_dataset.csv`.

---

## 2. Злиття та очищення даних (`merge.py`)

Оскільки оригінальний датасет з Kaggle містить багато пошкоджених рядків (NaN координати, обрізані масиви пікселів), цей скрипт проводить жорстку фільтрацію та об'єднання даних.

### Як використовувати:
1. Покладіть оригінальний датасет (`training.csv`) та вашу власну вибірку (`my_custom_dataset.csv`) поруч зі скриптом.
2. Запустіть `merge.py`.
3. Скрипт виконає наступні дії: Синхронізує назви колонок, видалить усі рядки з пропущеними координатами (Drop NaN) та перевірить цілісність кожної фотографії.
4. На виході ви отримаєте ідеально чистий файл `merged_dataset_clean.csv`, готовий для навчання нейромережі.

---

## 3. Генерація датасету емоцій (`extract_keypoints.py`)

Скрипт автоматично проганяє фотографії з емоціями через навчену нейромережу та формує готовий набір числових ознак (features) для навчання класифікатора емоцій (ЗНМ1).

### Як використовувати:
1. Покладіть вашу навчену ONNX-модель у теку зі скриптом.
2. Додайте датасет з обличчями (наприклад, теку `CK+48`, де всередині є підтеки з назвами емоцій).
3. Запустіть скрипт `extract_keypoints.py`.
4. Скрипт автоматично знайде обличчя, зробить нормалізацію та через `onnxruntime` отримає 30 координат від навченої моделі.
5. На виході ви отримаєте файл `emotion_keypoints.csv`, де кожному рядку відповідає мітка емоції (клас від 0 до 6) та 30 нормалізованих координат ключових точок.
